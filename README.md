# TCGA_BRCA_Dashboard
Interactive dashboard exploring tumor purity and survival in TCGA-BRCA

This dashboard explores the relationship between tumor purity, 
radiotherapy treatment, and overall survival in breast cancer 
patients using the TCGA-BRCA dataset.

## Live Dashboard
[View Dashboard](https://xian703.github.io/TCGA_BRCA_Dashboard)

## Why It Matters
Understanding how tumor purity interacts with radiotherapy response 
in breast cancer can help guide personalized treatment decisions. 
This dashboard provides researchers an interactive tool to explore 
survival heterogeneity across purity and treatment subgroups.

## Source
- Data: TCGA-BRCA via GDC Data Portal
- Tools: R, flexdashboard, plotly, DT
